Bank management using c Programming

This C program simulates a banking system where users can create accounts, log in, check balances, transfer money, and pay bills. Users can set up a banking PIN and transaction PIN during account creation. The program allows for logging in with a username and password, with options for checking balance, transferring money, and paying bills for services like DTH, mobile recharge, and electricity. Transactions require valid account numbers and a transaction PIN for security. The program also initializes a default admin account with preset credentials. The system handles multiple account interactions and updates balances accordingly after each transaction.

Navi bot using IOT

The Arduino Uno R3 microcontroller serves as the main control unit, coordinating the system's functions for efficient navigation. Bluetooth beacons provide real-time positioning, enabling the system to guide users along the most accessible and optimal route. The voice module (APR33A3) delivers clear directional commands, ensuring users receive accurate navigation instructions, while the ultrasonic sensor detects obstacles and halts the robot to ensure safety.
